# 🐍 Cursor Snake Animation (Pygame)

A visually smooth and interactive snake animation built with **Pygame**, where the snake follows your mouse cursor using vector-based movement and segment spacing logic.

---

## 🛠️ Technologies Used

* Python 3
* Pygame
* Math (trigonometry for movement and angles)

---

## 🎮 How It Works

* The **head** of the snake moves toward the mouse using trigonometric calculations.
* Each **body segment** follows the one ahead of it while maintaining a fixed distance.
* This creates a smooth, natural motion similar to **inverse kinematics**.
* A glow layer is drawn beneath each segment for visual enhancement.

---

## 💡 Inspiration

This project demonstrates how simple math and rendering techniques can create smooth and visually appealing animations in Python.
