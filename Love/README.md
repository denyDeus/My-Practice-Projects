# ❤️ Te Amo Particle Heart Animation

A simple and romantic web animation project that creates a beautiful **“Te Amo” love message using small heart particles**.
The hearts fly from the sides of the screen, form the text **“Te Amo”**, and then rearrange themselves into a **big heart shape**.

This project was created with love and creativity as a special dedication.

## ✨ Features

* 💘 Heart particles shooting from left and right
* 💘 Particles forming the text **“Te Amo”**
* 💘 Transformation into a large heart shape
* 💘 Smooth canvas animation
* 💘 Romantic dark gradient background
* 💘 Pure **HTML, CSS, and JavaScript (Canvas API)**

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Canvas API

---

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder in **VS Code**.
3. Make sure the following files exist:

   * `index.html`
   * `style.css`
   * `script.js`
4. Right-click `index.html`
5. Click **Open with Live Server**

The animation will start automatically in your browser.
